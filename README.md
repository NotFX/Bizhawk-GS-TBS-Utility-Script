# Bizhawk-GS-TBS-Utility-Scripts
The scripts we've gotten used to on VBA, but for Bizhawk's mGBA core instead.


How to use: 

1)Load Bizhawk

2)Load mGBA core

3)Load either GS:TBS or GS:TLA

3)Open LUA console

4)Load utility script

5)Follow console instructions
